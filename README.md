# Email-Spam-Classification
This project is a Machine Learning based Spam Email Classifier that can distinguish between spam and non-spam (ham) messages. It uses Natural Language Processing (NLP) techniques for text preprocessing and a Naive Bayes model for classification.

- Feature
  - Preprocessing email text (tokenization, stopword removal, etc.)
  - Training using Bernoulli Naive Bayes
  - Predicting whether an email is Spam or Ham
  - Streamlit-based web app for easy interaction

- Tech Stack
  - Python
  - Scikit-learn
  - NLTK
  - Streamlit
